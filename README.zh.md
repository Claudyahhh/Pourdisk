# Pourdisk

Pourdisk 是一个安全优先的 Codex skill，用来审计并清理 macOS 本地存储空间。

它适合在 Mac 存储空间告急时使用：先找出真正占空间的目录，再判断哪些可以安全删除，避免误删个人文件、聊天记录或项目资料。

## 工作流程

Pourdisk 遵循五步流程：

1. **审计**
   检查磁盘可用空间，并扫描主要存储区域，包括 `~/Library/Caches`、`~/Library/Containers`、`~/Library/Application Support`、开发者缓存，以及常见聊天和办公应用数据。

2. **分级**
   按风险对清理候选项分类：
   - 绿色：可重新生成的缓存和日志
   - 黄色：应用缓存或离线数据，删除后可能需要重新加载
   - 红色：个人文件、聊天历史、本地文档，必须明确审核后才可处理

3. **确认**
   向用户展示路径、大小、风险等级和删除影响。用户明确批准前，不执行删除。

4. **清理**
   只删除用户确认过的路径。skill 会避免宽泛的破坏性命令，并默认保护用户数据。

5. **复查**
   重新检查磁盘空间和相关应用目录，并说明释放了多少空间、首次打开应用时可能出现什么变化。

## 内置脚本

审计脚本是只读的：

```bash
./scripts/macos_storage_audit.sh
```

它会输出 Markdown 风格的存储报告，方便 Codex 基于报告生成清理方案。

## 默认安全边界

Pourdisk 默认不会删除：

- 桌面、文档、下载、照片或 iCloud Drive
- 源码仓库
- Obsidian vault
- Mail 或 Outlook 邮件库
- 微信 `msg` 文件夹或本地聊天历史
- 任何看起来像用户自己创建的应用数据

核心原则很简单：先审计，清楚分级，删除前确认，清理后复查。
